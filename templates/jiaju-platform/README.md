# 家具售卖平台 UI 模板

## 项目简介

这是一个家具售卖平台的静态 HTML 页面模板，包含完整的首页设计，适合用于家居、家具、装修类网站的前端开发参考。

## 技术栈

- HTML5
- CSS3
- 纯静态页面（无 JavaScript 依赖）

## 功能特点

- 响应式导航菜单
- Banner 轮播区域
- 产品展示区
- **3D 魔方照片展示效果**（CSS3 动画）
- 新闻资讯区
- 页脚信息

## 项目结构

```
jiaju-platform/
├── 家具售卖平台z.html    # 主页面
├── css/
│   ├── base.css          # 基础样式
│   ├── css.css           # 主样式
│   └── reset.css         # 样式重置
├── fonts/
│   └── iconfont/         # 图标字体
├── images/               # 图片资源
└── js/                   # JavaScript（预留）
```

## 使用方式

1. 直接用浏览器打开 `家具售卖平台z.html` 即可预览
2. 根据需要修改 CSS 样式和图片资源
3. 可集成到 Vue/React 等前端框架中使用

## 亮点功能

### 3D 魔方照片展示

页面中包含一个 CSS3 实现的 3D 魔方照片展示效果：
- 自动旋转动画
- 鼠标悬停展开效果
- 纯 CSS 实现，无需 JavaScript

## 适用场景

- 家具电商网站
- 家居装修平台
- 室内设计展示
- 前端学习参考
